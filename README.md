# Guide

Linux Electron App build and Snapcraft Release
## Installation

clone the repo 
```bash
git clone https://github.com/Efshal/boilerplate-monorepo.git
```
run npm install
```bash
npm install
```
run snapcraft init to setup snap/snapcraft.yaml
```bash
snapcraft init
```
